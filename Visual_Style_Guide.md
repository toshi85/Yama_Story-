# 🎥 山岳ドキュメンタリー 映像制作ガイド (Visual Style Guide)

AI動画生成（Runway/Luma）による「インパクト」と、静止画＋キャラ（Midjourney/Live2D）による「解説」を組み合わせ、視聴者を飽きさせないハイブリッド構成を作成するためのガイドライン。

## 1. ハイブリッド構成ルール
各シーンは原則として以下の2点セットで記述する。

1.  **【AI動画】**: 3〜5秒のインパクト映像。シーン転換や衝撃的な瞬間に使用。
2.  **【背景静止画】**: 動画の後に続く、ナレーター解説用の高画質背景。キャラを配置するスペースを考慮する。

## 2. プロンプト記述ルール (日本語)
編集者が直感的に理解し、生成ツールに入力しやすい具体的日本語で記述する。

### 構成要素
`[被写体] + [動き/状態] + [環境/照明] + [画質/スタイル]`

### 例: 遭難シーン
*   **【AI動画】**: 猛吹雪の中、動けなくなってうずくまる登山者。風で服が激しくたなびいている。視界不良。暗い青色のトーン、映画のような画質。
*   **【背景静止画】**: 猛吹雪の雪山。ホワイトアウト寸前の視界。手前に岩場の空きスペースあり（キャラ配置用）。寒々しい青白い照明。

## 3. シーン別キーワード集

### A. 遭難・絶望 (Despair)
*   **照明:** 薄暗い、月明かり、懐中電灯の光線
*   **環境:** ホワイトアウト、強風で舞う雪、鋭い岩場
*   **スタイル:** 手ブレのあるカメラ、GoPro風、ノイズ

### B. 平穏・回想 (Nostalgia)
*   **照明:** 夕暮れの黄金色（マジックアワー）、木漏れ日
*   **環境:** 青空、緑の草原、暖かいリビング
*   **スタイル:** フィルム粒子、ソフトフォーカス

### C. 捜索・緊迫 (Tension)
*   **照明:** ヘリのサーチライト、赤色回転灯、強いコントラスト
*   **環境:** 夜の山岳、指令室のモニター
*   **スタイル:** 空撮ドローン、ズームイン

## 4. 台本記述フォーマット
```markdown
【制作メモ】
*   **【AI動画】** (3秒): [ここに動画生成用プロンプト]
*   **【背景静止画】**: [ここに背景生成用プロンプト]
```
